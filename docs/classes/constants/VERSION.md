# Documentation - Urls::VERSION
## Description
The version of URLS Framework.
|  Type  |
| ------ |
| String |
## Examples
```PHP
<!-- blog-home.php -->
<!DOCTYPE html>
<html>
<head>
   <meta charset="utf-8">
   <title>My Blog - Home</title>
</head>
<body>
<h1>This is my Blog</h1>
<p>Welcome!</p>
<p>Powered by URLS Framework version: <?php echo htmlspecialchars(Urls::VERSION); ?></p>
</body>
</html>
```
